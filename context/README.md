# context/

Project context and collateral — the thinking, planning, and presentation material
developed alongside the MCP server. **Not** loaded at runtime and excluded from the
`.mcpb` bundle. This is the "why" and "where it's going," separate from the runtime
code at the repo root.

| Path | What it is |
|------|------------|
| `DEVELOPMENT_PLAN.md` | The API analysis + roadmap the server was built from. |
| `Zuar-Portal-MCP-Overview.md` | High-level overview of the MCP and its capabilities. |
| `Zuar-Portal-MCP-Automation.md` | Notes on automation / workflow / loop patterns. |
| `Zuar-Portal-MCP.pptx` | Presentation deck. |

For runtime code see `src/`; for user-facing documentation see `docs/`; for the
Claude Code agent ecosystem see `.claude/`.
